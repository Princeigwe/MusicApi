# MusicApi
A simple music API written with Django Rest Framework

## Authentication
It uses the Djoser package for user authentication.
JWT and Session are the modes of authentication.

## Documentation
API documentation can be found with the URL links:
- [Swagger](https://music-api-app.herokuapp.com/swagger).
